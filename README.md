# Classification of News Headlines

This project builds a machine learning model to categorize news headlines into five categories using linguistic features such as n-grams, POS tags, sentiment, and word embeddings. Our model uses multinomial logistic regression.

## Requirements

- Python
- Install dependencies:

```bash
pip install -r requirements.txt
```

Run Instructions: 
1. cd into src and run main.py
2. download dataset from: https://www.kaggle.com/datasets/setseries/news-category-dataset/data
